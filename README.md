# Créez un tableau de bord dynamique avec Power BI

## Contexte et objectifs

Ce projet consiste à créer un tableau de bord interactif permettant de suivre l’avancement de projets, identifier les retards et proposer une vision claire de l’état des tâches.

## Soft Skills
- Traduire un besoin métier en solution visuelle  
- Organiser l’information de manière lisible  
- Raconter une histoire avec les données  
- Améliorer l’accessibilité visuelle  

## Compétences
- Modélisation de données sous Power BI  
- Création de mesures DAX  
- Construction d’un visuel Gantt dynamique  
- Mise en place de filtres, interactions et KPIs  

## Outils utilisés
- Power BI Desktop  
- DAX
